📘 Panduan Cepat PM89

Alur Kerja

  1. Input data di tab Home (pakai waktu UTC)
  2. Sistem otomatis hitung delay & konversi ke local
  3. Klik Save → data masuk ke Data Approval
  4. Generate surat di tab Request Approval

Auto-Convert UTC ↔ Local

Saat kamu isi Route (contoh CGK-DPS), sistem ambil UTC offset dari tab Parameter. Di bawah setiap kolom UTC akan muncul preview waktu lokal otomatis.

  • CGK = UTC+7, DPS = UTC+8, UPG = UTC+8
  • Ubah STO di Parameter jika ada bandara baru

PM89 Threshold

  • ■ 0-60 menit: belum eligible
  • ■ 61-120 menit: warning
  • ■ 121-180 menit: PM89 mulai
  • ■ >240 menit: prioritas tinggi

Tips

• Selalu input dalam UTC untuk menghindari salah hari.
• Time Publish OCL wajib diisi untuk audit.
• Data otomatis terhapus setelah 7 hari.
